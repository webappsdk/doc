Detailed Description

Contains methods for parsing a property file and getting the parsed properties.

Example of property file: 

            ############################
            # EXAMPLE OF PROPERTY FILE #
            ############################
            # comment on property1
            property1=value1

            # comment on property2
            property2=value2
